글씨 연습장 생성기 사용 도움말

원하는 글자나 문장을 입력하면 A4 출력용 글씨 연습장을 자동으로 만들어주는 페이지입니다.

1. 이 페이지는 무엇인가요?

이 페이지는 사용자가 입력한 글자나 문장을 바탕으로 A4 용지에 출력할 수 있는 글씨 연습장을 만들어주는 도구입니다.

한글 따라쓰기, 문장 필사, 받아쓰기 준비, 영어 단어 쓰기 연습 등에 사용할 수 있습니다.

A4 출력 세로형 A4 용지에 맞춰 출력됩니다.
10칸 자동 정렬 기본값은 한 줄 10칸입니다.
5번 반복 기본값은 같은 내용을 5번 반복합니다.

2. 기본 사용 순서

  1. 제목 칸에 연습장 제목을 입력합니다.
  2. 연습 문장 칸에 따라 쓸 글자나 문장을 입력합니다.
  3. 필요하면 한 줄 칸 수, 반복 횟수, 글자 크기, 글씨체를 조절합니다.
  4. 미리보기 적용 버튼을 눌러 결과를 확인합니다.
  5. A4 인쇄 버튼을 눌러 출력합니다.

3. 제목 입력하기

제목 칸에는 출력물 상단에 표시될 이름을 입력합니다.

예시 펜글씨 연습 한글 따라쓰기 바른 글씨 연습 문장 쓰기 연습

4. 연습 문장 입력하기

연습 문장 칸에 따라 쓸 글자나 문장을 입력합니다.

여러 문장을 입력하고 싶을 때는 줄을 바꿔서 입력하면 됩니다.

입력 예시 봄여름가을겨울 하늘땅 별바람 나는 바르게 글씨를 씁니다

각 줄은 각각 다른 연습 문장으로 처리됩니다.

5. 긴 문장은 어떻게 출력되나요?

기본값은 한 줄 10칸입니다. 긴 문장을 입력하면 10칸씩 나누어진 뒤, 각 묶음이 반복됩니다.

입력 예시 가나다라마바사아자차카타파하 출력 방식 가 나 다 라 마 바 사 아 자 차 가 나 다 라 마 바 사 아 자 차 가 나 다 라 마 바 사 아 자 차 가 나 다 라 마 바 사 아 자 차 가 나 다 라 마 바 사 아 자 차 카 타 파 하 카 타 파 하 카 타 파 하 카 타 파 하 카 타 파 하
긴 문장은 전체 문장을 5번 반복하는 방식이 아니라, 10칸 단위로 5번 반복한 뒤 다음 내용으로 넘어갑니다.

8. 글씨체 사용 안내

기본 글씨체를 선택하거나, 내 PC에 설치된 글씨체 이름을 직접 입력할 수 있습니다.

  • 바탕체: 기본 글씨 연습용
  • 궁서체: 또박또박한 글씨 느낌
  • 맑은 고딕: 깔끔한 글씨 연습
  • 명조 계열: 책 글씨 느낌
  • 영문 기본체: 영어 단어 연습용
  • 내 PC 글씨체 직접 입력: 설치된 글꼴 이름을 직접 입력
내 PC 글씨체는 사용자의 컴퓨터에 설치되어 있어야 적용됩니다. 글꼴 이름이 정확하지 않으면 기본 글씨체로 보일 수 있습니다.

9. 따라쓰기 농도 안내

옵션 추천 상황
첫 줄 진하게, 아래로 갈수록 흐리게 처음 연습하는 학생에게 추천합니다.
전체 같은 흐림 모든 줄을 같은 농도로 따라 쓸 때 사용합니다.
첫 줄만 제시하고 나머지는 빈칸 익숙해진 뒤 스스로 써보는 연습에 좋습니다.

10. A4 인쇄 방법

  1. A4 인쇄 버튼을 누릅니다.
  2. 인쇄 창에서 용지 크기를 A4로 설정합니다.
  3. 방향은 세로로 설정합니다.
  4. 배율은 100%로 설정합니다.
  5. 칸 선이 흐리게 보이면 배경 그래픽 옵션을 켭니다.

12. 자주 생기는 문제

문제 해결 방법
출력물이 잘려요. 인쇄 배율을 100%로 설정하고, 용지 크기가 A4인지 확인하세요.
칸 선이 잘 안 보여요. 인쇄 설정에서 배경 그래픽 옵션을 켜보세요.
글씨체가 바뀌지 않아요. 해당 글씨체가 컴퓨터에 설치되어 있는지 확인하세요.
글자가 너무 커요. 글자 크기를 줄이거나 칸 높이를 키워보세요.
A4 한 장을 넘어가요. 문장 수를 줄이거나 반복 횟수를 줄여주세요.

13. 추천 사용 방법

  • 처음 연습할 때는 첫 줄 진하게, 아래로 갈수록 흐리게 옵션을 추천합니다.
  • 익숙해진 뒤에는 첫 줄만 제시하고 나머지는 빈칸 옵션으로 연습하면 좋습니다.
  • 짧은 단어는 한 줄 10칸을 추천합니다.
  • 긴 문장은 자동으로 10칸씩 나누어지므로 그대로 입력해도 됩니다.
완성된 연습지는 A4 용지로 출력하여 글씨 연습, 받아쓰기 준비, 문장 필사 연습 등에 사용할 수 있습니다.
